# Version Control Labs 

This repository contains my work for the Version Control course lab. The purpose of this lab was to understand and apply fundamental Git commands, manage a local repository, and sync it with GitHub using SSH.

## Lab Objectives:
* Generate an SSH key and link it to a GitHub account.
* Initialize a local Git repository and connect it to a remote GitHub repository.
* Practice staging and committing changes.
* Manipulate commit history using commands like `reset`, `amend`, and `revert`.
* Resolve merge conflicts locally.

## Tasks Completed:
1. **Initial Setup:** Created a file containing my name, made the first commit, and pushed it to GitHub.
2. **Commit History Manipulation:**
   * Made multiple commits to track changes.
   * Successfully deleted the last two commits using `git reset --hard`.
   * Created a new commit after the deletion.
3. **Commit Amending:** Appended text to a file and combined it with the previous commit using `git commit --amend`, updating the commit message to `"finish"`.
4. **Reverting (Bonus):** Applied the `git revert` command to undo specific previous commits and successfully resolved the resulting merge conflicts.

## Technologies Used:
* Git (Version Control System)
* Git Bash (Command Line Interface)
* GitHub
